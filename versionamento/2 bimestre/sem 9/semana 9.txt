Atividade B
1. Qual é a principal diferença entre Canary Release e Blue-Green Deployment?

Resposta: Canary Release libera a nova versão para poucos usuários primeiro. Já o Blue-Green Deployment utiliza dois ambientes separados para realizar a troca de versões com mais segurança.


1. Em qual situação o rollback é importante?

Resposta: O rollback é importante quando uma nova versão apresenta erros e a equipe precisa voltar rapidamente para a versão anterior.

1. Por que secrets são importantes para a segurança de uma aplicação?

Resposta: Porque protegem senhas, chaves e tokens, evitando que informações sensíveis fiquem expostas.

1. Docker e Kubernetes fazem a mesma coisa? Explique de forma simples.

Resposta: Não. Docker cria e empacota os containers, enquanto Kubernetes gerencia e controla vários containers em execução.

1. GitHub Actions é uma ferramenta de automação. Cite uma tarefa que ela pode executar automaticamente.
Resposta: Executar testes automáticos sempre que um novo código for enviado ao GitHub.
